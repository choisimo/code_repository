# 네이버 지도 API를 활용한 날씨 정보 웹앱 - Nodove

## 프로젝트 소개
이 프로젝트는 네이버 지도 API와 공공데이터 포털의 날씨 API를 활용하여 날씨 정보를 시각화하는 웹 애플리케이션입니다.

## 기술 스택

### 프론트엔드
- Vue.js 3 (Vue CLI)
- Vue Router
- Pinia (상태 관리)
- Axios
- 네이버 맵 JavaScript API
- SCSS / Tailwind CSS
- Glassmorphism UI 디자인

### 백엔드
- Java 21
- Spring Boot 3.x
- Spring Web
- Spring Data JPA
- Spring Data MongoDB
- Spring Data Redis
- Spring Data Redis Reactive
- Spring Boot WebFlux
- Spring Boot WebSocket
- Spring AI OpenAI
- MariaDB
- Lombok
- Gradle
